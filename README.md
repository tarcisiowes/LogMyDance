# LogMyDance

Mobile app for logging and tracking dance sessions.

Built with [Expo](https://expo.dev) (bare workflow), React Native, and React.

## Stack

- **Expo** 56
- **React Native** 0.85.3
- **React** 19

## Getting Started

```bash
npm install
```

### Android

```bash
npm run android
```

### iOS (macOS only)

```bash
npm run ios
```

### Web

```bash
npm run web
```
