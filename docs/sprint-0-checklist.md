## P0 — Must complete before Sprint 1

- [ ] Create lo-fi prototypes for the 5 key flows:
  - [ ] Add a class entry
  - [ ] Log a new movement
  - [ ] Browse movement library
  - [ ] Use a class template
  - [ ] Find a movement from a past class
- [ ] Test prototype with 5–10 dancers (at least 3 in-person sessions)
- [ ] Survey beta testers on their devices (Android model / iOS version)
- [ ] Download and use DanceDay for 1 week, take notes on UX gaps
- [ ] Download and use Dance Journal for 1 week, take notes on UX gaps

## Prototype Feedback Template

Interview each dancer with these questions:

1. When you leave a dance class, what do you wish you had written down?
2. How do you currently track movements you're learning?
3. What would make you open this app the day after a class?
4. What would make you stop using it after 2 weeks?
5. Show prototype — talk-aloud: "What do you expect to happen if you tap here?"
