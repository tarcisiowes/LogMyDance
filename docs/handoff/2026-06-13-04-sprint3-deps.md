## Blockers
- ⚠️ `react-native-zip-archive` is a **native module** → JS-only reload will NOT
  pick it up. A native rebuild is mandatory before export/import work on device.

## Manual actions required
1. `npx expo prebuild` (only if `android/` `ios/` are managed/regenerated) — repo
   ships `android/gradlew` so the Android project exists; a clean prebuild is the
   safe path if the build fails.
2. Rebuild native app:
   - Android: `npx expo run:android`
   - iOS: `pod install` then `npx expo run:ios`
3. `npx expo install --check` to confirm versions.
4. First `expo start` after this regenerates typed-routes — removes the need for the
   `as Href` cast on the new `/storage` route (see handoff 07). Cast is harmless
   until then.
