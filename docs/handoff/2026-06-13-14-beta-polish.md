# 14 — Beta polish (Sprint 5, codeable parts)

**Date:** 2026-06-13
**Status:** ✅ Done — `tsc --noEmit` clean.
**Sprint:** 5 (Beta)

## What was done
The in-app pieces that support a manual-retention beta (roadmap Option A — user shares their own stats; no invasive analytics).

- **Share diagnostics**: new card in the Stats → Beta Diagnostics section. Builds a plain-text payload (version, language, entries, movements, practice minutes, current/longest streak, app opens, days-since-install) and opens the OS share sheet (`Share.share`). No PII — only counts.
- **Install-date tracking**: `preferences.ensureInstallDate()` (set once on first launch, from `AppStartup`) + `getInstallDate()`; powers `days_since_install`.
- **Settings → Beta section**:
  - *Send feedback* → `mailto:` (recipient from `extra.feedbackEmail`, empty = blank compose).
  - *Reset onboarding* → clears the MMKV flag and returns to `/onboarding` (handoff 10 follow-up, useful for testers).
