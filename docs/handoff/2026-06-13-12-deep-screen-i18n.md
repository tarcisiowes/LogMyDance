## Decisions / gotchas
- Validation: dropped zod message strings; render `errors.x ? t('forms.xRequired') : undefined` (each form has a single required field).
- Count strings use plain `{{count}}` + literal `(s)` rather than i18next plural suffixes (keeps typed keys simple; acceptable for beta). i18next returns the base key and interpolates when no `_one/_other` variant exists.
- Dance-style names (Bachata, Salsa‚Ä¶) are **not** translated ‚Äî proper nouns, and they're DB-seeded data.
